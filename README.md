# 🏭 Spring Boot — Factory Design Pattern

A backend Java application built with Spring Boot that demonstrates the Factory Design Pattern with a clean and modular project structure.

## Features
- Implementation of the Factory Design Pattern
- RESTful API endpoints to demonstrate object creation via factory
- Clean separation of interface, concrete classes, and factory logic

## Tech Stack
- Java, Spring Boot, Maven

## How to Run
1. Clone the repository
2. Open in IntelliJ IDEA or Eclipse
3. Run as a Spring Boot application
4. Test endpoints via Postman or browser at `http://localhost:8080`

## Design Pattern Overview
The Factory Pattern is used to create objects without specifying the exact class. This project demonstrates how Spring Boot can be structured around this pattern for scalable and maintainable backend design.
